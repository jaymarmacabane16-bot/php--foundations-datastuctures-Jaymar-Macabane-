# php--foundations-datastuctures-Jaymar-Macabane-

Project Purpose

A simple PHP library system that organizes and displays books using arrays and functions. It shows how to view, search, and sort books by category or title.

Personal Information

Purpose: Displays a person’s basic details using an associative array.

How to Run: Open the PHP file in a browser.

Example Output:
Name: Juan Dela Cruz  
Age: 20  
Address: Davao City  
Hobby: Playing Volleyball

 Book Information

Purpose: Shows book details like author, year, and genre using arrays.

How to Run: Run the PHP file to see book info displayed.

Example Output:
Title: Treasure Island  
Author: Robert Louis Stevenson  
Year: 1883  
Genre: Adventure


Library Category Browser

Purpose: Displays books in categories and subcategories using nested arrays.

How to Run: Run the PHP file and view the category list.

Example Output:
Adventure  
    Classic Adventures  
        Treasure Island  
        Journey to the Center of the Earth



Complete Library Management System

Purpose: Combines all features — book list, details, category view, and search.

How to Run: Run the PHP file to see categories, book details, alphabetical list, and search results.

Example Outputs:
Search: "Dracula" → Found  
Search: "Moby Dick" → Not Found  

Alphabetical List:  
Dracula  
Pride and Prejudice  
Treasure Island
